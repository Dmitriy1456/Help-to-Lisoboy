# 🕸️ Async Web Crawler & Text Analyzer

![Python Version](https://img.shields.io/badge/python-3.12+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Асинхронное консольное приложение для автоматического сбора данных с веб-ресурсов. Проект разработан в рамках изучения принципов сетевых соединений, протокола HTTP и асинхронной обработки данных.

## 🎯 Цели проекта
- [x] Изучить протокол HTTP и работу с сессиями.
- [x] Освоить парсинг HTML (BeautifulSoup4).
- [x] Реализовать многопоточную обработку через `asyncio`.
- [x] Применить алгоритмы текстовой аналитики.

---

## ⚙️ Функционал
- **Глубокое сканирование:** Обход домена с ограничением уровня вложенности.
- **Аналитика:** Подсчет частоты слов и фильтрация «мусорных» данных.
- **Анти-блокировка:** Имитация реального пользователя (User-Agent, Referer, Delays).
- **Экспорт:** Формирование структурированного `JSON` отчета.
